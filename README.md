# Basic Login Page

Testing functionality of login Page


### Prerequisites

* JDK 15
* IDE (IntelliJ OR Eclipse)

### Running the tests

* ( testng.xml ) can be found in the root path of the project
* ( Command Line ) mvn clean test 
     mvn clean test -Dbrowser={browserType}
